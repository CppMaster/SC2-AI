# Welcome to SC2-AI

<p> Welcome to SC2-AI a program built to play Star Craft 2 an implimentation of ALPHAStar</p>

***

### Setup

Create virtual env
#### Mac/Linux
```
python3 -m venv venv
```

#### Windows
```
py -3 -m venv venv
```

### Install Virtual env
```
pip install -r requirements.txt 
```
